16)Запись прав доступа проводится следующим образом:
-rwxrwxrwx => у пользователей нет доступа к файлу без специальных прав.
-rwxr-xr-x => позволяет другим пользователям просматривать файлы, но при этом сторонние пользователи не могу ни перезаписывать информацию в файлы, ни удалять, ни создавать новые.
-rwx------ => польностью закрывается доступ ко всем файлам в каталоге для сторонних пользователей.

Изменение прав доступа проводится с помощью команды chmod
