1) Node.js представляет среду выполнения кода на JavaScript, которая построена на основе движка JavaScript Chrome V8, который позволяет транслировать вызовы на языке JavaScript в машинный код.

Node.js прежде всего предназначен для создания серверных приложений на языке JavaScript. Хотя также существуют проекты по написанию десктопных приложений (Electron) и даже по созданию кода для микроконтроллеров.

*******************************************************************************************

2) Express - это фреймворк для Node.js, который реализовывает слой функций, необходимых для создания эффективных приложений и API. Его использование значительно сокращает написание кода, а, значит, уменьшается затрачиваемое на разработку время.

*******************************************************************************************

3) MongoDB - это документоориентированная NoSQL база данных.

MongoDB работает быстрее, обладает лучшей масштабируемостью, ее легче использовать в отличии от реляционных баз данных.

Вся система MongoDB может представлять не только одну базу данных, находящуюся на одном физическом сервере. Функциональность MongoDB позволяет расположить несколько баз данных на нескольких физических серверах, и эти базы данных смогут легко обмениваться данными и сохранять целостность.

Для хранения данных в MongoDB применяется формат, который называется BSON (БиСон) или сокращение от binary JSON.

MongoDB хранит документы. В отличие от строк, которые хранятся в реляционных базах данных, документы могут хранить сложную по структуре информацию. Документ можно представить как хранилище ключей и значений.

С учётом плюсов и минусов всех баз данных, следует понимать, что для разных типов данных подойдут разные БД. Можно комбинировать базы данных для хранения разной информации.
